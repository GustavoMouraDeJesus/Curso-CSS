Resumindo, os prinicipais seletores que temos são esses:

  2. Temos também o seletor através do identificador, que vamos colocar um id e oque tiver aquela class do nosso id no css receberá aquelas propriedades CSS.
  3. E o terceiro que utilizamos para dar propriedade para todo nossa página nos utilizamos um coringa, que seria o *{}, isso dentro da nossa tag css, isso vai dizer que todas as tags do nosso projeto receberam as propriedades dentro desse coringa.

Conclusão

Os seletores CSS são essenciais para a estilização de páginas web, permitindo aplicar estilos de forma eficiente e organizada. Aprendemos que existem diferentes tipos de seletores, como:

  1. Seletor de tag – Aplica estilos a todas as ocorrências de uma determinada tag.
  2. Seletor de classe – Permite definir estilos reutilizáveis e aplicá-los a múltiplos elementos usando a propriedade class.
  3. Seletor de ID – Específico para um único elemento, definido com a propriedade id.
  4. Seletor universal (*{}) – Aplica estilos a todos os elementos da página.

Além disso, vimos que é possível agrupar seletores para otimizar o código e evitar repetições desnecessárias. Também destacamos a importância das regras de nomenclatura, como evitar iniciar identificadores com números e garantir que cada id seja único. Com esse conhecimento, podemos estruturar e personalizar nossos sites de maneira mais eficiente, garantindo um design mais organizado e de fácil manutenção.
